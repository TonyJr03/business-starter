# Icons compartidos

Directorio para íconos SVG de uso global (no específicos de ningún negocio).

Ejemplos de archivos aquí:
- `whatsapp.svg`
- `instagram.svg`
- `facebook.svg`
- `tiktok.svg`
- `youtube.svg`
- `map-pin.svg`
- `clock.svg`
- `phone.svg`

> Por ahora el proyecto usa íconos inline en componentes Astro o emoji como
> placeholders de íconos. Mover aquí cuando se establezca un sistema de íconos
> SVG dedicado.
