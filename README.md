# Business Starter

Starter modular para crear sitios web de negocios locales con **Astro + React + Tailwind CSS + Supabase + Vercel**.

## Descripción

Este proyecto nace con el objetivo de construir una base reutilizable para desarrollar sitios web de negocios, comenzando por **cafeterías en Cuba**.

La idea es ofrecer una plantilla moderna, rápida y personalizable que permita lanzar sitios para distintos negocios sin reconstruir todo desde cero cada vez.

El starter está pensado para cubrir necesidades comunes como:

- presencia web profesional,
- información del negocio,
- catálogo o menú,
- promociones,
- contacto y ubicación,
- gestión operativa básica,
- módulos opcionales como carrito y pedidos por WhatsApp.

## Objetivo

Construir una plantilla modular y escalable para sitios de negocios, empezando por cafeterías, con enfoque inicial en proyectos por encargo y visión futura de evolución hacia un SaaS multi-negocio.

## Stack tecnológico

- **Astro** — estructura del sitio, rendimiento y SEO
- **React** — interactividad en módulos específicos
- **Tailwind CSS** — sistema de estilos y UI reusable
- **Supabase** — base de datos, autenticación y backend ligero
- **Vercel** — despliegue

## Enfoque inicial

La primera etapa del proyecto se enfoca en:

- construir un **starter único**,
- clonarlo y adaptarlo por cliente,
- permitir branding configurable,
- ofrecer catálogo, promociones y secciones informativas,
- permitir gestión operativa mediante un panel simple.

## Visión futura

A largo plazo, este proyecto busca sentar las bases para evolucionar desde un starter por cliente hacia una solución **SaaS multi-negocio**.
